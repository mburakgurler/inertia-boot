# Kotlin Spring Boot + InertiaJS PoC

This demo project demonstrates the proof of concept of InertiaJS with Kotlin Spring Boot.

## Features

- Kotlin Spring Boot
- Vue 3 + Vite 
- InertiaJS integration

## Prerequisites

- JDK 17
- Node.js 16+
